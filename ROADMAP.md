# Roadmap

StreetSpeak AI will ship in phases. Live trading must remain disabled by default and should only appear after mock, safety, confirmation, and audit paths are mature.

1. Repo foundation
2. Mock trading desk
3. Local voice command flow
4. Order ticket builder
5. Safety and confirmation engine
6. Local audit log
7. Robinhood MCP read-only adapter
8. Robinhood order review
9. Live equity execution with strict confirmation
10. Desktop app wrapper
11. Public adapter
12. Options ticket parsing
13. Redacted proof/audit cards
