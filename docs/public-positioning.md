# Public Positioning

StreetSpeak AI is an open-source, voice-native trading desk for AI agents and self-directed trading workflows. The v0.1 demo is local-first, mock-first, and built to show safety gates before any future broker execution work.

## What It Is

- A local mock trading desk for typed and optional browser-native voice commands.
- A transparent order-ticket and confirmation demo for self-directed users.
- A redacted local audit timeline and receipt export workflow.
- A fixture-only Robinhood readiness panel for future read-only data shape review.

## What It Is Not

- Not investment advice.
- Not a broker, adviser, signal service, or recommendation engine.
- Not live trading software in v0.1.
- Not connected to Robinhood, Public, ElevenLabs, or a live market data API.
- Not an autonomous or scheduled trading system.

## Core Safety Message

StreetSpeak AI starts with mock mode locked on. Order-like commands create mock tickets only. Generic confirmations are rejected. Mock submission requires the exact challenge phrase and code. Receipts and audit exports state that no live broker order was placed.

## Robinhood Readiness Message

Robinhood read-only support is future-gated. The current Robinhood panel is fixture-only and static. It shows no live connection, requires no credentials, and has no order review, order placement, cancel order, or live execution capability.

Future phases must stay separate:

1. Read-only MCP connection.
2. Order review.
3. Live execution, only if separately approved.
