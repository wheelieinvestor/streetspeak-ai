# Collaboration Checklist

## Daily Workflow

- Pull latest main.
- Create a branch.
- Make a small change.
- Test locally.
- Push branch.
- Open PR.
- Request review.
- Merge only after approval and CI pass.

## Before Opening PR

- Branch is up to date with main.
- No secrets committed.
- Tests pass locally.
- Build passes locally.
- README/docs updated if needed.

## Before Merging

- At least 1 approval.
- CI passing.
- Conflicts resolved.
- PR description is clear.
- Squash merge selected.
